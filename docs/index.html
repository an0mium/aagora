aragora

A society of heterogeneous AI agents that discuss, critique, improve each other's responses, and learn from successful patterns.

PyPI v0.7.0 GitHub MIT License Python 3.10+
pip install aragora

What's New in v0.7.0

  • Graph-based debates with branching
  • Evidence provenance chains
  • Consensus memory & dissent retrieval
  • Scenario matrix debates
  • Executable verification proofs
  • Red-team adversarial mode
  • Human intervention breakpoints
  • Self-improvement code editing

⚙ Core Features

Heterogeneous Agents

Mix Claude, Codex, Gemini, Grok, Qwen, Deepseek, and local models in a single debate.

Structured Debate

Propose, Critique, Revise loop with configurable rounds and consensus mechanisms.

ELO Rankings

Agents are ranked by performance. Better agents get selected for harder tasks.

Memory Streams

Agents maintain long-term memory of past debates and learned patterns.

🔮 Advanced Capabilities New

Debate Graph

Non-linear debates with counterfactual branching. Explore alternative reasoning paths in parallel.

Evidence Provenance

Cryptographic chain tracking evidence sources. Merkle tree verification for integrity.

Consensus Memory

Store debate outcomes. Retrieve similar past debates and dissenting views.

Scenario Matrix

Run debates across multiple scenarios (scale, risk, time horizon) in parallel.

Verification Proofs

Link claims to executable code. Verify assertions automatically with sandboxed execution.

Red-Team Mode

Adversarial testing with steelman/strawman attacks. Find weaknesses before production.

💻 More Features

Human Breakpoints

Pause debates for human input when confidence is low or stakes are high.

Domain Templates

Pre-built templates for code review, design docs, incident response, research.

Self-Improvement

aragora can edit its own code. Git integration for safe modifications.

GitHub Action

Multi-agent PR reviews. Drop into any repo's CI/CD pipeline.

Typed Claims

Structured reasoning with claim types, evidence categories, and logical relations.

Adaptive Selection

Automatically select the best agents for each task based on expertise and history.

$ aragora ask "Design a rate limiter for 1M req/sec" --agents codex,claude [Round 1] codex_proposer: Proposing solution... [Round 1] claude_critic: Analyzing proposal... Issues found: Race condition in distributed counter [Round 2] codex_proposer: Revising based on critique... [Round 2] claude_critic: Re-evaluating... Consensus reached (confidence: 87%) $ aragora demo rate-limiter # Quick 60-second demo $ aragora templates # List available debate templates
